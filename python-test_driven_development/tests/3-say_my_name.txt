The ``3-say_my_name`` module
======================

Using ``say_my_name``
-------------------

This is an doctest file for say_my_name function.
The function takes two arguments (first_name and last_name) and output text in the format ``` My name is <first name> <last name>```. 
If an argument is not an string, a TypeError is raised.

First import ``say_my_name`` from the ``3-say_my_name`` module:

    >>> say_my_name = __import__('3-say_my_name').say_my_name

Now use it:

    >>> say_my_name("John", "Smith")
    My name is John Smith
    >>> say_my_name("Walter", "White")
    My name is Walter White
    >>> say_my_name("Bob")
    My name is Bob

A TypeError is raised if first_name or last_name are not strings:

>>> say_my_name(12, "White")

>>> say_my_name("Zenitsu", 12)
